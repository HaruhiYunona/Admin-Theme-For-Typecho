# Admin-Theme-For-Typecho


自用的Typecho后台皮肤,需要替换文件。使用前请注意备份文件。
全粉圆角设计,少女心满满。

**使用方法:**将该压缩包解压以后放入你的admin目录即可。如果提示文件重复请选择直接替换。
